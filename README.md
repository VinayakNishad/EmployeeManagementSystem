# EmployeeManagementSystem
A React + Node.js Employee Management System that allows users to manage employee records, filter/search employees, and export employee data to CSV. The app provides an intuitive UI with pagination, filtering, and CRUD (Create, Read, Update, Delete) functionalities with pagination.


Features

Add / Edit / Delete Employees — Manage employee details dynamically.
Search and Filters — Search employees by name or email and filter by department, designation, or joining date.
CSV Export — Export all employee records as a CSV file with properly formatted date values.
Pagination — Navigate through employee lists easily using Previous/Next buttons.
Responsive UI — Built using React-Bootstrap for smooth experience across devices.
API Integration — Fetch employee data dynamically using Axios from backend API (http://localhost:3001/api/employees).
